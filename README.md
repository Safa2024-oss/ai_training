AI Vulnerability Detection Training
Project Overview

This project focuses on training an Artificial Intelligence (AI) model to detect potential security vulnerabilities in source code.
The goal is to help developers and students identify risky code patterns such as SQL Injection, Cross-Site Scripting (XSS), and other common vulnerabilities.

The trained model can later be integrated into a security scanner or development tool to analyze code and provide explanations of detected vulnerabilities.

Objectives

Build a dataset of vulnerable and safe code samples.

Train a machine learning model to classify vulnerabilities.

Evaluate the performance of the model.

Save the trained model for integration into a security analysis system.
